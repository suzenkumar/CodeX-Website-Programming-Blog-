What is Html ?

1. Introduction — What is HTML?

HTML stands for HyperText Markup Language. It is the standard markup language used to create and structure content on the World Wide Web.

  • Provides the skeleton of a web page
  • Browsers read HTML and render the structured content visually
  • Works together with CSS (style) and JavaScript (interactivity)

2. Brief History

  • Early 1990s — Tim Berners-Lee created the first version of HTML.
  • 2000s — HTML evolved; XHTML introduced.
  • 2014 — HTML5 became W3C standard.
  • Today — HTML5 is maintained as a living standard.

3. Basic Structure of an HTML Document

<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <title>Page Title</title>
    <meta name="viewport" content="width=device-width, initial-scale=1">
  </head>
  <body>
    <h1>Hello, world!</h1>
    <p>This is a simple HTML page.</p>
  </body>
</html>

4. HTML Elements and Tags

HTML uses elements, defined by tags:

<p>This is a paragraph.</p>

Example of attributes:

<a href="https://example.com" target="_blank">Visit</a>
<img src="photo.jpg" alt="A photo" width="400">
